# Event Managment as a Event Creator

Learn how to manage your events as a event creator effectively on our RSVP Event SaaS platform with this comprehensive guide.

Welcome to our RSVP Event SaaS platform! This guide will walk you through the process of manage your event step by step.

## Step 1: Navigate to the Event Management Page
1. Go to **My Events** page.
2. Click on the **Manage Event** button, present on the each events.
3. You will be redirected to the event management page.

## Step2: Event Management Page Overview
1. On the event managment page, you will see following:
    - **Event Title** : A clear and descriptive name for your event.
    - There are three sections on the event management page:
        - **Overview**: There is brief event name, event start date and time, event end date and time, event location, event description.
        - You can **Edit Event** button, when clicked you will be navigated to edit event page  and **Share Event** button when clicked, the link of the event will be copied and same link you can share to others.
        - On the same page you can see recently registered attendees and their status with registration date.
        - You can also add you cohosts, spcial guests and event managers of the event.
        - Then we have another section called **Guests** section where you can see the list of all the guests who have registered for the event.
        - You can download the guest list in excel format.
        - You can help guests checkin to the event by scanning the QR code of the guest.
        - Then we have third section called **Communications** section where you can send messages to the guests who have registered for the event.
        - Then we have fourth section called **More** section where you can change the public URL of the event to share it with people and you can also cancel the event.
